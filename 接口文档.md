# 个人中心界面接口文档

### 问卷列表
- param:{userName:String} //post请求，分页和排序功能在前端已实现
- respond:
  ```
  //可能的话按照时间顺序排序
  {quizs[{  
        name:String,  
        ID:String,
        //是否发布
        state:Boolean,
        date:Date,
        //回收量
        num:Int,
        },{}......]
  }
  ```
